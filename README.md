# 小新智能网站生成器

一个基于 AI 的智能网站生成平台，支持通过自然语言描述快速生成各类网站应用。

## 项目简介

小新智能网站生成器是一款 AI 驱动的网站代码生成平台，用户可以通过聊天对话的方式描述需求，AI 会自动生成对应的网站代码。目前支持生成：

- **单页 HTML**：简单的单页面网站
- **多文件代码**：包含 HTML + CSS + JavaScript 的完整网页
- **Vue 项目**：完整的 Vue 3 项目代码

## 技术栈

### 后端

- Java 21
- Spring Boot 3.x
- MyBatis-Flex (数据库 ORM)
- Redis + Redisson (分布式缓存)
- Sa-Token (认证授权)
- LangChain4j (AI 代码生成)
- Spring Doc OpenAPI (API 文档)

### 前端

- Vue 3 + TypeScript
- Vite (构建工具)
- Ant Design Vue (UI 组件库)
- Pinia (状态管理)

## 功能特性

- 🤖 **AI 智能生成**：通过对话描述需求，AI 自动生成代码
- 💬 **流式输出**：支持 SSE 流式响应，实时展示生成过程
- 📱 **应用管理**：创建、编辑、删除、部署应用
- 📂 **代码下载**：一键打包下载生成的项目代码
- 🌐 **在线部署**：生成的应用可直接部署预览
- 📊 **截图功能**：自动生成应用预览截图
- 👥 **用户管理**：完整的用户认证和权限管理
